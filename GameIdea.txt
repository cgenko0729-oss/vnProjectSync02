1. VNDirector 复合演出编排器 💎💎💎💎 🔧中（第三次推荐，它依然是最值的）
- 把现有 API 固化成一行名场面：Confession()（樱吹雪+心跳+缓推+黄昏色调）、Nightmare()（荷兰角+恐怖色调+危险泛光+强震）、WakeUp()（眨眼转场+清晨色调+光束渐亮）、FlashbackStart/End()（白闪+回忆色调+Bokeh+云影）。
- 附带演出队列：Director.Queue(...) 顺序播放多个演出，为剧本系统铺路。

2. 台词自动演出（Text-Driven Performance） 💎💎💎💎 🔧中
- 在台词字符串里嵌标记，对话框播放时自动触发演出："什么？！{shake}{snapzoom}你说她失踪了？"——打字机走到标记处自动执行震动/急推/泛光/表情。这是把 60 个 API 真正"喂"给剧本作者的管道，写剧本就是在导演。

3. 待机彩蛋系统 💎💎 🔧低
- 玩家 15 秒无操作：角色随机小动作（现有 Nod/HeadShake/Shy 随机播）+ 镜头极缓漂移；一有输入立刻归位。画面永远"活着"。

4. 视线跟随鼠标 💎💎 🔧低
- 立绘随鼠标位置轻微倾斜/平移 1~2%（视差的立 ，像角色在"注视"你的光标方向。梦幻系互动感。
                                                                                                        二、光影细节层（下一个"高级感"金矿）

5. 窗格光影投射（Gobo Shadow） 💎💎💎 🔧中
- 在背景上叠一层"窗框/树叶间隙"形状的光斑遮罩（程序化格子/叶隙图案 + 乘法混合），随时间极缓慢移动。教室午后、树荫下的经典氛围，和 God Rays 是绝配。                          
6. 烛光/篝火闪烁光照 💎💎💎 🔧低
- 一个"火光源"组件：全屏暖橙叠加层 + 立绘 EmissionAmount 以火焰噪声曲线（不规则闪烁，非正弦）同步抖动 + 边缘泛光同步。夜晚篝火/烛光晚餐场景一键成套
- 同理可做闪电照亮：雷暴时立绘和背景瞬间被打亮一帧。

7. 雷暴组合演出（VNThunderstorm） 💎💎💎 🔧低
- 纯编排现有件：随机间隔 → 白闪半程 + 0.3s 后中震 + 雨量瞬时加倍 + 立绘瞬间打亮。做成常驻组件挂着就自动"下暴雨"。

8. 开关灯瞬间 💎💎 🔧低
- LightsOff()：0.15 秒内全画面压暗到 0.25 + 暗角收紧（黑暗中只看到轮廓）；LightsOn() 反向 +
微白闪。停电/深夜回家的演出。

9. 玻璃雨滴流淌 💎💎 🔧高
- 雨天"贴着窗户看外面"：全屏叠加雨滴挂壁+蜿蜒流下的折射 shader。效果极佳但要写专门的 droplet shader，放低优先级。

                                                                                                        10. 剪影登场（Silhouette Reveal） 💎💎💎 🔧

11. 说话节奏点头（Talking Bob） 💎💎 🔧低（遗留）
- 打字机播放时说话立绘随文字节奏轻微起伏，字停人停。与说话者高亮联动后，"角色在说话"的错觉完成度极高。
                                                                                                        12. 泪光效果 💎💎 🔧中
- 悲伤场景：立绘脸部区域（uv 上半的可配置小区域）叠加 1~2 个缓慢下滑的高光亮点 + 星光闪烁。配合 Sadness 泛光和沮丧动作成套。
                                                                                                                       13. 立绘姿态交叉溶解 💎💎💎 🔧中
- 同角色换表情/换姿势立绘时，旧图溶解出+新图溶解入（或交叉淡化 0.3s），而不是瞬间跳变。做剧本系统前的必备基础件。
